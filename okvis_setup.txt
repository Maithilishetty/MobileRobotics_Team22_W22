OKVIS installation:

1) Install all the dependencies mentioned on github: https://github.com/ethz-asl/okvis

2) Build the project
	cd okvis
	mkdir build && cd build
	cmake -DCMAKE_BUILD_TYPE=Release ..
	make -j8


3) If you get errors running make, you'll probably have to replace all CV_BGR2GRAY with cv::COLOR_BGR2GRAY and all CV_AA with cv::LINE_AA, in:
	okvis/okvis_multisensor_processing/src/VioVisualizer.cpp
	okvis/okvis_apps/src/okvis_app_synchronous.cpp
	maybe some other places too
	
4) To download the example data, make sure you are one directory below the okvis folder, and run:
	wget http://robotics.ethz.ch/~asl-datasets/ijrr_euroc_mav_dataset/machine_hall/MH_01_easy/MH_01_easy.zip
	unzip MH_01_easy.zip && rm MH_01_easy.zip

5) To run on the example data, run the following from the build directory:
	 ./okvis_app_synchronous ../config/config_fpga_p2_euroc.yaml ../../mav0

6) To run on the cave data, run the following from the build directory (make sure the data exists and is correctly configured):
	 ./okvis_app_synchronous path/to/okvis/config/config_cave.yaml ../../modified_cave
